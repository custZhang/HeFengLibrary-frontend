### 本地运行
部署前端工程前，请先确保后端已经部署完成

1. 拉取项目
2. 安装项目所需依赖
```yaml
npm i
```

3. 运行项目
```yaml
npm run serve
```

4. 访问http://localhost:8201，就能访问到该项目
